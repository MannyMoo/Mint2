Detailed Description

this implements: class Neg2LL: public Minimisable

in a way that its constructor

Neg2LL( pdf, eventlist, MinuitParameterSet* mps=0)

swallows any type of pdf or eventlist as long as:

o pdf implements IPdf<EVENT_TYPE> o eventlist as the method size() and the operator [unsigned int i] o eventList must return an event type that pdf can digest.

This specific setup below with Neg2LLClass (defined first, don't use) and Neg2LL (defined 2nd) is chosen such that you can declare Neg2LL w/o the need to specify template arguments, i.e. like this:

Neg2LL fcnP(BpPdf, BpEventList);

It will take any PDF and Eventlist, with the restrictions described above. This is a bit friendlier than old MINT Neg2LL, which required that you specify template arguments as in Neg2LL<EVENT_TYPE> fcnP(...)
