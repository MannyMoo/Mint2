14 /* we assume the following format:
15 
16 ParName iFix Par0 Step Pmin Pmax ::
17 
18 Ex 1 0. 1e-4 0.0 0.0
19 Ey 1 0. 1e-4 0.0 0.0
20 
21 Where Pmin and Pmax can be left out;
22 
23 Iff the number of parameters read in after the parameter name
24 is 6, however, we assume the following format:
25 
26 ParName N iFix Par0 Step Pmin Pmax ::
27 
28 Ex 1 1 0. 1e-4 0.0 0.0
29 Ey 2 1 0. 1e-4 0.0 0.0
30 
31 This includes the parameter number N. This is meaningless for us.
32 But it makes it compatible with Mikhail's input files. However,
33 this will not work if you specify N, but leave out any of the other
34 fields - this is only detected by counting the number of fields
35 following the parameter name!!
36 */
